# FeedCast Radio - Landing Page

A modern, premium landing page for FeedCast Radio - AI-powered personalized radio.

## Features

- **Dark theme design** inspired by Spotify and Apple
- **Fully responsive** - works on all devices
- **Interactive audio player** with waveform visualization
- **Smooth animations** and micro-interactions
- **Waitlist signup** form
- **No dependencies** - vanilla HTML, CSS, and JavaScript

## Structure

- `index.html` - Main landing page
- `styles.css` - All styles with modern CSS features
- `script.js` - Interactive functionality and animations
- `assets/` - SVG icons and favicon
- `README.md` - This file

## Design Philosophy

This landing page embodies the premium quality of FeedCast Radio:

1. **Premium feel** - Dark gradients, smooth transitions, quality typography
2. **Interactive elements** - Animated waveforms, playable sample player
3. **Clear value proposition** - Hero message immediately communicates the product
4. **Social proof ready** - Structured for adding testimonials and user counts
5. **Conversion optimized** - Strategic CTAs and waitlist capture

## Key Sections

1. **Hero** - "Your world, narrated" with animated background
2. **How It Works** - 3-step process visualization
3. **Pricing** - Free vs Premium comparison with toggle
4. **Sample Player** - Interactive audio player with waveform
5. **Waitlist** - Email capture with benefits
6. **Footer** - Clean minimal footer

## Technical Notes

- Uses Inter font family for modern typography
- CSS Grid and Flexbox for responsive layouts
- Canvas-based waveform visualization
- Intersection Observer API for scroll animations
- Local storage ready for user preferences
- Form validation and feedback

## Deployment

This is a static site that can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting provider

Simply upload all files maintaining the directory structure.

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile browsers (iOS Safari, Android Chrome)
- Progressive enhancement for older browsers
